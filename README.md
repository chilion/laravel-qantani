# Qantani
A connector to the Qantani Payment API for usage with Laravel or Lumen

# Usage
## composer
"CJSDevelopment/qantani" : "dev-master"

## app.php
### ServiceProvider
'CJSDevelopment\QantaniServiceProvider'

### Facade
'Qantani' => 'CJSDevelopment\Qantani',
